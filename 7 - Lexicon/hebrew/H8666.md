# H8666 תְּשׁוּבָה / תשובה

## tᵉshûwbâh

_(tesh-oo-baw' | teh-shoo-BAW | teh-shoo-VA)_

### Definition

or תְּשֻׁבָה; from H7725; a recurrence (of time or place); a reply (as returned); answer, be expired, return; n-f

- a recurrence, an answer, return
  - return
    - completion of a year, return of a year
  - at the return (construct)
  - answer, reply
