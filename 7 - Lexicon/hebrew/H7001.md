# H7001 קְטַר / קטר

## qᵉṭar

_(ket-ar' | keh-TAHR | keh-TAHR)_

### Definition

from a root corresponding to H7000; a knot (as tied up), i.e., (figuratively) a riddle; also a vertebra (as if a knot); doubt, joint; n-m

- knot, joint, problem
  - joints (of the hip)
  - difficulties, doubts (figuratively)
