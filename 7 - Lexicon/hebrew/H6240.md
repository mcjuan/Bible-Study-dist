# H6240 עָשָׂר / עשר

## ʻâsâr

_(aw-sawr' | aw-SAWR | ah-SAHR)_

### Definition

for H6235; ten (only in combination), i.e., -teen; also (ordinal) -teenth; (eigh-, fif-, four-, nine-, seven-, six-, thir-)teen(-th), + eleven(-th), + sixscore thousand, + twelve(-th); n

- ten, -teen (in combination with other numbers)
  - used only in combination to make the numbers 11–19
