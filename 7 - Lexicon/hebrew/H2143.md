# H2143 זֵכֶר / זכר

## zêker

_(zay'-ker | dzay-KER | zay-HER)_

### Definition

or זֶכֶר; from H2142; a memento, abstractly recollection (rarely if ever); by implication, commemoration; memorial, memory, remembrance, scent; n-m

- memorial, remembrance
  - remembrance, memory
  - memorial
