# H6212 עֶשֶׂב / עשב

## ʻeseb

_(eh'seb | eh-SEB | eh-SEV)_

### Definition

from an unused root meaning to glisten (or be green); grass (or any tender shoot); grass, herb; n-m

- herb, herbage, grass, green plants
