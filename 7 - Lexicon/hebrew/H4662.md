# H4662 מִפְקָד / מפקד

## miphqâd

_(mif-kawd' | mip-KAWD | meef-KAHD)_

### Definition

from H6485; an appointment, i.e., mandate; concretely, a designated spot; specifically, a census; appointed place, commandment, number; n-m

- muster, appointment, appointed place
  - muster (of people)
  - appointment
  - appointed place
- number
